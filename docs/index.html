Part I
Basic Usage

1 Arming and Disarming

Arming is performed by pressing CTRL + ALT + A. A tray message will confirm arming. You can only arm LaptopAlarm while you are logged in to Windows.

Be aware that arming the alarm while the power cord is removed or the battery is disconnected (if enabled in settings) will immediately trigger the alarm.

By default, the alarm is triggered when the laptop’s power cord is disconnected.


Figure 1: Arming tray message

Disarming is performed by pressing CTRL + ALT + D. A tray message will confirm disarming. You can only arm LaptopAlarm while you are logged in to Windows. To prevent others from disarming LaptopAlarm, it is recommended that you have a password lock on your user account and lock (WINDOWS + L) to lock your account.


Figure 2: Disarming tray message

2 Cancelling an alarm

Should an alarm be triggered, an “ALARM” dialog box will show and a tray notification will be displayed. To cancel the alarm, you must be signed into your account (you cannot be on the Windows lock screen). Then, press CTRL + ALT + D or disarm the alarm from the context menu.


Figure 3: Alarm tray notification


Figure 4: Alarm dialog box

3 Tray Icon Context Menu

Right clicking LaptopAlarm’s tray icon will show a context menu. From this menu, you can arm or disarm the alarm, trigger the alarm in panic situations, show LaptopAlarm’s configuration options, or exit LaptopAlarm.


Figure 5: Tray Icon Context Menu

3.1 Triggering Alarm

You can immediately trigger the alarm from the right-click context menu. You can use this feature in case of panic. You can remove this option from the context menu from the “Program Settings” tab in the settings window.

Part II
Customizing Settings

4 Accessing the Settings Window

To access the LaptopAlarm settings window, right-click the LaptopAlarm tray icon and select “Show”. LaptopAlarm’s settings page will show.


Figure 6: Settings window

5 Alarm Settings

5.1 Alarm Triggers

This section alters when LaptopAlarm will trigger the alarm.

Power cord If this option is selected, LaptopAlarm will trigger the alarm when the power cord is disconnected. Be aware that arming the alarm when the power cord is disconnected will cause an alarm.

USB Device If this option is selected, LaptopAlarm will trigger the alarm whenever any USB device is disconnected.

Battery removal If this option is selected, LaptopAlarm will trigger the alarm whenever the laptop’s battery is removed. Be aware that arming the alarm when the battery is removed will cause an alarm.

Computer restart If this option is selected, LaptopAlarm will trigger the alarm whenever the computer is restarted. LaptopAlarm must be configured to startup on computer startup for this to happen (see “Program Settings” tab).

6 On Alarm

6.1 Play Audio

Sounds You can select the sound that LaptopAlarm will play when the alarm is triggered. The default sound is a burglar alarm sound. You may specify a custom sound by selecting “Custom Sound” and choosing a .wav file.

Increase Volume on Alarm LaptopAlarm, by default, will increase and lock the computer volume at 80%. You may specify a different audio volume, or turn off the feature entirely.

Test alarm Clicking this button will sound the alarm. This button will not increase the volume.

6.2 Send email to self

This feature has not been tested. If this option is enabled, LaptopAlarm will send an email to the email address you specify.

SMTP Settings These are settings provided by your email provider. Check with them for more details.

7.1 Start program on computer restart

This option does not work. It is currently being worked on.

7.2 Show “Trigger Alarm” in Context Menu

If this option is selected, the option “Trigger Alarm” is shown in the right-click context menu. By default, this option is selected.

7.3 Automatically send bug and crash reports

If this option is selected, LaptopAlarm will automatically send bug and crash reports to the developer when one is encountered.
